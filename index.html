A landing page template freshly redesigned for Bootstrap 4!

About This Page

This is a great page to talk about your webpage. This template is purposefully unstyled so that you can use it as a boilerplate or starting point for your own landing page designs. This template features:

  • Clickable nav links that smooth scroll to page section
  • Responsive behaviour when clicking nav links perfect for a one page website
  • Bootstrap's scrollspy feature which highlights which section of the page you are on the navbar
  • Minimal custom CSS so you arefree to explore your own unique design options
